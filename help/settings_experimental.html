Experimental

These limits protect DC++ from excessive memory, connection, and packet use. The defaults are recommended for normal use. Lower values can reject legitimate traffic; higher values increase resource usage and exposure to denial-of-service traffic. Changes apply to new operations without restarting DC++.

Queued protocol data limit
Maximum unsent protocol data retained for one socket. The socket is disconnected when another write would exceed the limit. Minimum: 1024 bytes. Default: 16777216 bytes.
Concurrent peer connection limit
New incoming peer connections are rejected when this many incoming and outgoing transfer or private-message connections already exist. Hub connections are not included. Minimum: 1. Default: 512.
Incoming connection flood window
Time span used to count incoming peer connection attempts from each IP address. The allowed burst is derived from the multi-connection upload setting. Minimum: 1 millisecond. Default: 10000 milliseconds.
Global UDP rate window
Time span used by the global incoming search-result UDP packet limiter. Minimum: 1 millisecond. Default: 1000 milliseconds.
Global UDP packet limit
Maximum incoming search-result UDP packets processed during the global UDP rate window. Minimum: 1. Default: 2000.
Per-peer UDP packet limit
Maximum incoming search-result UDP packets processed from one IP address during the per-peer UDP rate window. Minimum: 1. Default: 500.
Per-peer UDP rate window
Time span used to count incoming search-result UDP packets separately for each IP address. Minimum: 1 millisecond. Default: 10000 milliseconds.
Tracked UDP peer limit
Maximum number of IP addresses retained by the per-peer UDP limiter. New addresses are ignored while the table is full and no expired entries can be removed. Minimum: 1. Default: 4096.
Encrypted UDP packet limit
Maximum encrypted search-result UDP packet accepted for decryption. Valid packets must also be at least 32 bytes and aligned to a 16-byte AES block. Minimum: 32 bytes. Default: 4096 bytes.
Stored encrypted UDP key limit
Maximum recent search encryption keys retained for decrypting SUDP results. Oldest keys are discarded first. Minimum: 1. Default: 64.
Generated partial file list limit
Maximum uncompressed XML size generated in response to a partial file-list request. Minimum: 1024 bytes. Default: 67108864 bytes.
